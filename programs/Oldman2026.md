### Principles

Main design principles:

1. Each main lift gets one "heavy" session per week.
2. Each workout targets different muscle groups (no "leg day" or "chest day"). 
3. Overall volume per muscle group should be moderately high, but most of it should come from lower weights. 
4. Use very low (conservative) starting weights for all exercises, including pullups (by doing assisted pullups) and other supporting exercises.
5. Pseudo-linear progression for main lifts based on RPE, maybe periodization in the future.

Note:

1. Target RIR (Reps In Reserve) is really supposed to be an __upper limit__ since I'll be far away from it in the beginning. The target RIR is meant to be for the __last__ working set. 
2. Dumbbell weight is the weight total, so each dumbbell is half of that.
3. Pull-ups and chin-ups are meant to be assisted or weighted. The weight is `bodyweight + added weight - assisted weight`.

### Progression

Rule of thumb: 2 tiers of weights to add per exercise, a "large increment" and a "small increment". Do:

1. Add "large increment" if last week `Target RIR + 2 < Estimated RIR`, 
2. Add "small increment" if last week `Target RIR <=  Estimated RIR < Target RIR + 2`.
3. 0 if last week `Estimated RIR < Target RIR` or I did not make the target number of reps. 
4. Decrease by "large increment" if in situation 3 for two weeks in a row.

I expect it will take ~3 months before getting to challenging weights for the heavy exercises, somewhat shorter for some of the assistance work. At that point, I might consider things like deload or periodization but for now that's a distraction.
